# Biblia Bot w JavaScript - wersja 1

## **Instalacja pakietów**

* Wpisz w terminalu po kolei następujące komendy:

```javascript
npm init
```

```javascript
npm install discord.js
```

``` javascript
npm install dotenv
```

``` javascript
npm install -D nodemon
```

## **Uruchomienie bota** 

* Wpisz w terminalu tę komendę:
   
``` javascript
npm run start:dev
```

## Informacje

* Bot zawiera Biblię Gdańską z 1881 roku
